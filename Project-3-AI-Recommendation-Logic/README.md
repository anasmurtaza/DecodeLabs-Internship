# 🎯 Project 3 - AI Recommendation Logic

A professional **AI Recommendation System** built for the **DecodeLabs Artificial Intelligence Internship**.

This project creates personalized recommendations based on user preferences using content-based similarity logic.

---

## 🚀 Project Goal

Create a simple recommendation system based on user preferences.

---

## ✅ Key Requirements

- Take user input using choices or interests
- Match preferences using logic or similarity
- Display recommended items
- Build recommendation concepts through practical implementation

---

## ✨ Features

- Modern React + Vite frontend
- Multiple interest selection
- Dynamic recommendation generation
- Content-based similarity scoring
- Top-N recommendation control
- Match percentage display
- Attractive glassmorphism UI
- Animated colorful LED borders
- Responsive design
- GitHub-ready folder structure

---

## 🧠 Recommendation Logic

The system compares selected user interests with item tags.

It calculates a similarity score using:

- Tag overlap
- Union of user and item tags
- Preference boost
- Ranking from highest score to lowest score

The best matching items are displayed as personalized recommendations.
