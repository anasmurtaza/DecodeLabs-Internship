## рџ§  How It Works

The chatbot follows a simple rule-based approach:

1. User enters a message.
2. Input is converted to lowercase.
3. Extra spaces are removed.
4. The chatbot checks predefined rules.
5. If a match is found, it returns the corresponding response.
6. Otherwise, it displays a fallback message.
